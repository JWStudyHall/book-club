# Book Club App

A web app for creating and joining book clubs.

Deploy app [click here](book-club-online.up.railway.app)
## Features

- Create book clubs
- Join existing book clubs
- User accounts

## Tech Stack

- Frontend: HTML, CSS, JavaScript
- Backend: Node.js (optional)
- Database: Optional
